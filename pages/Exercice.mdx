
# Exercice Pratique : Gestion d'une Base de Données MongoDB

**Objectif :** Appliquer les concepts abordés dans le cours sur MongoDB en pratiquant la création, la gestion et la sécurisation d'une base de données.

## Partie 1 : Indexation

1. **Création d'une Collection :** Créez une collection appelée `utilisateurs` avec des documents représentant des utilisateurs. Chaque document doit avoir des champs tels que `nom`, `email`, `ville`, et `age`.

2. **Création d'un Index :** Appliquez l'indexation en créant un index sur le champ `ville` pour accélérer les recherches basées sur la ville.

## Partie 2 : Administration

3. **Sauvegarde et Restauration :** Utilisez la commande `mongodump` pour sauvegarder la collection `utilisateurs`. Ensuite, utilisez `mongorestore` pour restaurer la collection à partir de la sauvegarde.

4. **Monitoring des Performances :** Activez le profiler avec `db.setProfilingLevel(2)` pour collecter des informations détaillées sur les performances pendant une certaine période. Analysez les résultats pour identifier les opérations les plus coûteuses.

## Partie 3 : Sécurité

5. **Authentification et Autorisation :** Créez un utilisateur avec des privilèges spécifiques (par exemple, `readWrite` et `dbAdmin`) pour accéder à la base de données.

6. **Chiffrement des Données :** Activez le chiffrement au repos pour la base de données en utilisant une clé de chiffrement.

7. **Pare-feu MongoDB :** Configurez le pare-feu MongoDB pour autoriser uniquement une adresse IP spécifique à accéder à la base de données.

## Partie 4 : Bonus

8. **Audit des Opérations :** Activez l'audit pour surveiller les opérations de lecture et d'écriture dans la base de données. Consultez les journaux d'audit pour examiner les activités enregistrées.

## Instructions :

- Utilisez le shell MongoDB pour effectuer toutes les opérations.
- Documentez chaque étape et expliquez les raisons derrière vos choix.
- Soumettez un rapport décrivant en détail chaque partie de l'exercice, y compris les commandes utilisées, les résultats obtenus et les observations.

Cet exercice pratique vous permettra de mettre en œuvre de manière concrète les concepts enseignés dans le cours, couvrant les aspects d'indexation, d'administration et de sécurité dans MongoDB.
